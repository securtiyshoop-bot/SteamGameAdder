# 🚀 Hızlı Başlangıç

## 30 Saniyede Oyun Ekle

### 1. İndir & Çalıştır
```
1. SteamGameAdder.exe indir
2. Çalıştır
```

### 2. Bilgileri Doldur
```
Oyun Adı: [Oyunun Adı]
Klasör: [game.exe'nin olduğu yol]
App ID: [Steam ID]
Executable: game.exe
```

### 3. Ekle
```
[Tıkla: ➕ STEAM'E EKLE]
```

### 4. Bitir
```
Steam'i yeniden başlat → Oyun görunecek! ✅
```

---

## App ID Hızlı Bul

**Google'da ara:**
```
"[Oyun Adı] steam app id"
```

**Veya:**
https://steamdb.info → Oyun ara → ID kopyala

---

## Genel Hatalar

| Hata | Çözüm |
|------|-------|
| Executable not found | Klasörü kontrol et |
| Admin needed | Admin olarak çalıştır |
| Steam not found | Ayarlar → Steam yolunu gir |
| Oyun görünmüyor | Steam'i yeniden başlat |
