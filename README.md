# Steam Game Adder 🎮

**Professional Desktop Application** - Steam kütüphanesine herhangi bir oyunu direkt olarak ekleyin.

## 🌟 Özellikler

- 🎯 **Direkt Oyun Ekleme** - Tek tıkla Steam'e oyun ekle
- 📁 **Dosya Seçici** - Kolayca oyun klasörü seç
- 🖼️ **Icon Desteği** - Oyun için özel ikon ekle
- ⚙️ **Otomatik Yapılandırma** - Steam dizini otomatik bulunur
- 💾 **Oyun Listesi** - Eklediğin tüm oyunları sakla
- 🗑️ **Oyun Kaldırma** - Eklemiş oyunları kaldır
- 🌙 **Dark Theme UI** - Modern ve şık arayüz
- ⚡ **Hızlı İşlem** - Saniyeler içinde tamamlanır
- 🔧 **Ayarlar Paneli** - Steam yolunu manuel ayarla
- 📋 **Config Dosyası** - games.json ile oyun listesi kaydet

## 📥 Kurulum

### Option 1: Portable (Tavsiye)
```
1. SteamGameAdder-Portable.zip indir
2. Çıkart ve SteamGameAdder.exe çalıştır
3. Hazır!
```

### Option 2: Installer
```
1. SteamGameAdder-Setup.exe indir
2. Kur
3. Masaüstünden başlat
```

## 🚀 Kullanım

### Basit Adımlar:
1. **Oyun Adı Gir** - Örn: "GTA V"
2. **Klasör Seç** - Game.exe'nin olduğu klasörü seç
3. **App ID Gir** - Steam App ID (271590 = GTA V)
4. **Icon Seç** (Opsiyonel) - Oyun ikonu
5. **+ EKLE** butonuna tıkla
6. **✓ Steam'e eklendi mesajını** bekle
7. **Steam'i yeniden başlat** - Oyun görunecek!
