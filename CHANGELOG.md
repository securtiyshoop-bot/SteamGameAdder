# Changelog

## [1.0.0] - 2026-06-27

### Added
- ✅ Direkt Steam oyun ekleme fonksiyonu
- ✅ Oyun klasörü otomatik seçimi
- ✅ Executable otomatik algılaması
- ✅ Icon desteği
- ✅ Oyun listesi yönetimi
- ✅ Eklenen oyunları kaydetme (games.json)
- ✅ Oyun silme özelliği
- ✅ Steam dizini otomatik bulma
- ✅ Manual Steam yolu ayarları
- ✅ Modern dark theme UI
- ✅ Windows Forms integration (FolderBrowserDialog)
- ✅ Admin hak kontrolü
- ✅ Hata mesajları ve validasyon
- ✅ Status bar ile işlem gösterimi

### Fixed
- ✅ Steam manifest VDF formatı düzeltildi
- ✅ Path handling iyileştirildi
- ✅ Error messages Türkçe yapıldı
