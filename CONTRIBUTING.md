# Contributing

## 🎯 Katkıda Bulunmak İstiyorum?

Harika! Tüm katkılar hoşlanır. İşte nasıl başlayacağınız:

## 📝 Hata Bildirme

1. [GitHub Issues](https://github.com/securtiyshoop-bot/SteamGameAdder/issues) sayfasını aç
2. "New Issue" butonuna tıkla
3. Hatanızı açıkça yazın:
   - Ne yapmaya çalışıyordunuz?
   - Ne beklediniz?
   - Ne oldu?
   - Sistem bilgileri (Windows sürümü, .NET sürümü)

## 💡 Feature İsteği

1. [GitHub Issues](https://github.com/securtiyshoop-bot/SteamGameAdder/issues) sayfasını aç
2. "New Issue" butonuna tıkla
3. Feature talebinizi açıkla:
   - Ne eklemek istiyorsunuz?
   - Neden gerekli?
   - Nasıl kullanılacak?

## 🔧 Kod Katkısı

### Adım 1: Fork et
```bash
git clone https://github.com/YOUR_USERNAME/SteamGameAdder.git
cd SteamGameAdder
```

### Adım 2: Branch oluştur
```bash
git checkout -b feature/your-feature-name
```

### Adım 3: Değişiklikleri yap
```bash
# Kodunuzu yazın
# Testler ekleyin
```

### Adım 4: Commit et
```bash
git add .
git commit -m "Add your feature description"
```

### Adım 5: Push et
```bash
git push origin feature/your-feature-name
```

### Adım 6: Pull Request aç
1. GitHub'da repo sayfasına git
2. "Pull requests" tabına tıkla
3. "New Pull Request" butonuna tıkla
4. Değişikliklerinizi açıklayan mesaj yaz
5. "Create Pull Request" tıkla

## 📋 Kod Standartları

- C# standartlarına uyun
- Fonksiyonlara dokumentasyon yazın
- Test kodu yazın
- Türkçe yorumlar tercih edilir
